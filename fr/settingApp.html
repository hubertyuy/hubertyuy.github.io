Paramètres de l'application

Désinstaller une application, ou réinitialiser le profil et les paramètres de l'application

Sélectionnez une application pour afficher ses paramètres et la désinstaller ou l'arrêter.

Configurer les applis

Sélectionnez l'icône dans le coin supérieur droit de l'écran pour afficher le menu des détails.

Autoris. des applis Lorsque vous installez une application, vous pouvez contrôler les fonctions et informations auxquelles cette application peut accéder (ceci est appelé 《permissions》).
Liens de l'application Si le système détecte plusieurs applications qui peuvent effectuer la même opération (comme l'ouverture d'une connexion ou d'un fichier), vous serez invité à sélectionner une application.
Paramètres avancés
Applications par défaut Sélectionnez les applications par défaut de navigateur, d'assistance et de saisie vocale.
Se superposer aux autres applications Choisissez les applications qui peuvent être affichées au dessus d'autres applications que vous utilisez actuellement.
Modifier paramètres système Gérez les applications autorisées à changer les paramètres du système.
Optimisation de la batterie Économisez la batterie en gérant l'efficacité opérationnelle des différentes applications.

Menu des détails

Sélectionnez le menu des détails dans le coin supérieur droit de l'écran

Afficher les processus système Affichez ou cachez les applications du système
Réinitialiser préférences Supprimez les paramètres de limitation des données des applications et de l'arrière-plan. Tous les paramètres seront réinitialisés.

Désinstaller des applications

Sélectionnez une application et son écran de paramètres apparaîtra.
