Configurer Bluetooth

Vous pouvez utiliser fonction intégrée Bluetooth de cet appareil pour vous connecter sans fil à d'autres appareils (haut-parleurs, casques mains libres, écouteurs, claviers sans fil, etc.).

Après avoir activé la fonction Bluetooth, une liste des appareils à proximité qui prennent en charge Bluetooth s'affichera.

Pour préserver la charge, il est recommandé que vous désactiviez Bluetooth lorsque vous n'êtes pas connecté à un appareil compatible Bluetooth.

Se connecter à un appareil Bluetooth

La première fois que vous connectez cet appareil avec un autre appareil Bluetooth, les appareils doivent être appairés.

1Allumez l'appareil que vous souhaitez connecter grâce à Bluetooth, et réglez l'appareil sur un paramètre d'analyse. Pour obtenir des instructions sur la façon de permettre à un appareil de scanner, consultez le mode d'emploi de cet appareil.

2Sélectionnez le bouton 《rechercher les appareils》 dans le coin supérieur droit de l'écran.

3Sélectionnez le nom de l'appareil auquel vous souhaitez vous connecter dans le menu 《appareils disponibles》.

4Appuyez sur 《apparier》 et entrez le mot de passe (par exemple: 0000) pour se connecter à l'appareil Bluetooth.

Après avoir réussi à vous connecter une première fois à un périphérique Bluetooth, les appareils se connecteront automatiquement par la suite si Bluetooth est activé et les deux appareils sous tension.

Remarque:

Pour se reconnecter à un appareil appairé, sélectionnez le nom de l'appareil à reconnecter dans le menu 《Dispositifs Appairés》.
