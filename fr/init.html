Paramètres par défaut

L'écran affiche les paramètres par défaut lorsque l'appareil est allumé pour la première fois.

Veuillez modifier les paramètres en utilisant l'Assistant de Paramètres.

Les paramètres peuvent être modifiés à tout moment.

Remarque

Avant d'accéder aux paramètres par défaut, veuillez vous assurer que les paramètres du dispositif de partage sans fil (routeur sans fil, routeur portable MobileWiFi, etc.) permettront à cet appareil de se connecter.

1Définir les paramètres de langue.

Sélectionnez la langue de votre choix en faisant glisser vers le bas sur le côté gauche de l'écran.




Le système et la langue d'affichage apparaîtront dans la langue que vous définissez.

2Sélectionnez "commencer l'installation" pour entrer dans l'écran suivant.

3Choisir et utiliser le point d'accès Wi-Fi.




Saisir un mot de passe si le statut de l'onde électrique du point d'accès Wi-Fi affiche l'image d'une serrure. Une fois le mot de passe saisi, sélectionnez "se connecter".
