Activer 《Ne Pas Déranger》

Utiliser le paramètre 《Ne Pas Déranger》

Déroulez le menu rapide en faisant glisser votre doigt du haut de l'écran vers le bas,
puis sélectionnez 《Ne Pas Déranger》.


1Aucune interruption : désactivez tous les sons, y compris les alarmes
Spécifiez si vous souhaitez que ce paramètre reste activé pendant
une certaine période ou jusqu'à ce que vous le désactiviez.

2Alarmes uniquement : désactivez toutes les sonneries et le vibreur à l'exception des alarmes
Spécifiez si vous souhaitez que ce paramètre reste activé pendant
une certaine période ou jusqu'à ce que vous le désactiviez.

3Prioritaires uniquement : désactivez tous les sons, sauf pour certains appels ou notifications
Spécifiez si vous souhaitez que ce paramètre reste activé pendant
une certaine période ou jusqu'à ce que vous le désactiviez.

Activer le Mode Avion

Les fonctions sans fil de l'appareil (Wi-Fi, Bluetooth) seront désactivées une fois le ''mode Avion'' activé.

Convient lors d'un déplacement en avion ou dans des conditions similaires.

Activez via le menu des réglages rapides.

1Faites glisser la barre d'état située en haut de l'écran et sélectionnez 《Mode Avion》.

Activer via le bouton servant à mettre sous tension l'appareil.

1Maintenez enfoncé ce bouton.

L'écran affiche le menu d'arrêt de l'appareil.



2Sélectionnez 《Mode Avion》.

Remarque

En mode avion, vous pouvez toujours activer manuellement les fonctions
