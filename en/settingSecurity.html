Security Settings

Modify device’s security settings.

Device security
Screen lock Choose screen lock:
Swipe
Unlock your device by swiping your finger across the screen. Although anyone can unlock your screen, you can get to the home page screen quickly.
Pattern
Draw a pattern using your finger to unlock the device.
PIN
Input 4 digits to unlock the screen.
Password
Input a password of at least 4 letters or numbers to unlock the screen.
Lock screen message Input the characters you would like to display while the screen is locked.
Smart Lock
You can set a SmartLock to unlock the device during the following situations:
·Trusted devices
·Trusted places
·Trusted face
·Trusted voice
·On-body detection
Encryption
Encrypt tablet Add encryption to this device’s internal information. Once a device is encrypted, its previous settings cannot be restored. To restore previous settings, delete all data and format the system.
Passwords
Make passwords visible Check this box to make the characters typed into password boxes visible as you type them.
Device administration
Device administrators Display the device administrator.
Unknown sources Check this box to allow websites or emails from sources other than GooglePlay to install apps. To protect this device and your personal information, we recommend that users limit app installation permissions to GooglePlay and other reliable sources.
Credential storage
Storage type Hardware-backed
Trusted credentials Display credential information. Select a credential name to display its details.
System
Display this device’s pre-installed certificate authority (CA).
User
Display the CA set up by the user.
Install from SD card Install certificate authority information by connecting to an external USB hard drive or SD card.
Clear Credentials Delete all certificate information and certificates from the device’s memory.
Advanced
Trust agents Manage the apps which can access Smart Lock and other functions.
Screen pinning Turn this on to fix the screen which is currently being displayed on the device.
Apps with usage access Manage which apps can access the device’s app usage information.
