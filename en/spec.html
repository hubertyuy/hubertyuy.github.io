Tech specs

Operating System Android 6.0.1
Display 10.5” Super AMOLED
Resolution 2560 × 1600 (WQXGA)
Multi-touch functionality 10 points multi-touch
CPU Exynos 7420 Octa 2.1GHz, 1.5GHz
System Memory LPDDR4 3GB
Storage eMMC 64GB
Network Wi-Fi 802.11a/b/g/n MIMO 2x2
Location GPS, Glonass
NFC
Bluetooth 4.1
Sensors Ambient light sensor/Motion G-sensor/Compass sensor/Gyro sensor
I/O interface USB Type-C *1
Digital microphone
3.5 mm stereo audio jack
High Quality Speakers
Memory Slot: MicroSD card (up to 200 GB)*4
Camera Front camera: 5 Mega-Pixel
Main camera: 8 Mega-Pixel
Auto Focus
Battery 22.33Wh Li-Polymer (non-removable)
AC adapter AC-USB, Input:100-240V,Output: 5V/2.4A, 12V/1.5A(quick charge)*2
Dimensions 246.6 x 176 x 6.4mm
Weight 430g*3
  • *1 Please use the included USB 3.0 transfer cable when charging your device.
  • *2 This product does not come with an adapter. Please purchase one separately.
  • *3 The actual product weight may differ by ±5g.
  • *4 This device does not support exFAT format. Supported format: FAT32.
    Reading and writing are supported in FAT32 format (files and data folders can contain foreign letters and symbols).
