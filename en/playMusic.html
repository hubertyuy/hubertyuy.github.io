Playing music

Music files stored on the device can be played using the “Play Music” app.

Music can be played in the background while other apps are in use.

1Select the “Play Music” icon from the app menu.

2Album names will appear on the screen. Select the album or song you would like to listen to and touch the “play” button.

The device will begin playing the selected content.

Playing music in the background

“Play Music” can be used to play music while you are using other apps.

While music is playing in the background, pull down the notifications panel to pause a song or skip tracks.

Transfer music files from your computer to this device

Windows

1Connect this device to your computer using a USB Type-C transfer cable.

2Open this device’s internal storage folder using Windows Explorer, and place the music files in the “Music” folder.

Android supports MP3, M4A, AMR, MIDI, and OggVorbis music formats.

Mac OS X

As Mac OS X does not support Media Transfer Protocol (MTP), additional, free software must be installed.

1After theAndroid File Transferpage displays on the Mac, select Download Now to begin downloading the installation file.

2Open the downloaded file and follow the instructions on the screen to complete installation.

3Connect this device to the Mac using a Micro USB transfer cable.

4Open Android File Transfer.

5The screen will display this device’s folder contents. Place the music files you would like to add to this device in this device’s “Music” folder.

This device will automatically scan the added music files and display them in the Play Music app.
