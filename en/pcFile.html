Copying files from a computer

Files can be copied from a computer in four ways:

  • The files can be copied onto this device using a USB Type-C connection.
  • Files can be sent via Bluetooth.
  • Using the internet.
  • Using a microSD memory card.

Note

Downloaded files can be read and edited using an app that supports their filetypes.

Eg.) PDF：AdobeReader; JPEG: Photo Gallery; MP3: Play Music; Word/Excel: DocumentToGo

Using a USB connection

Please read“Connecting with a USB cable”.

Receiving files through Bluetooth

1The device must first be synched with the computer from which the files will be transferred.

For setup instructions, please read“Setting up Bluetooth”.

2Transfer files from the sending computer using the Bluetooth function.

3This device will send a notification when a file is received. To view the notification, open by notification panel by dragging the upper left side of the screen downward, then select the notification.

The screen will display a box asking you to accept the transfer.

4Select “accept” to receive the information.

Transferring files online.

GoogleDrive, Dropbox and other online services allow users to copy files (Word/Excel/PowerPoint documents, PDFs, images, music, etc) from their computer to this device using the internet.

1Download the GoogleDrive or Dropbox apps via the Play store.

2Upload files from your computer.

3Download the files onto this device using the app.

Using a microSD memory card.

This machine is equipped with a slot for microSD memory cards.

Note

MicroSD memory cards are an optional additional purchase.

MicroSD memory cards can serve as this device’s external memory (limited to 200GB), transferring information between computers, digital cameras, and other devices.

*Does not support exFAT format. Supported formats include FAT32/NTFS. While this device can read files in NTFS format (limited to files and folders saved using English/Arabic numerals), these files cannot be edited. Files in FAT32 format may be edited (file and folder names display non-English characters normally).

Installation

Installation

1Insert the card ejector tool into the hole beside the SIM card tray, until the tray comes loose. Then, pull the tray outward.

2Ensure that the microSD card is facing the proper direction.

3Push the card tray into the KALOS2, until the card tray is secure.

Ensure that the protective cover is closed, without any cracks.

Removal

Removal

1Insert the card ejector tool into the hold beside the SIM card tray, until the tray comes loose. Then, pull the tray outward.

2After removing the microSD card, push the card tray into the KALOS2, until the card tray is secure.

Note

Do not remove the microSD card while it is writing or reading information.
