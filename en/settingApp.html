Setting apps

Uninstall an app, or reset profile and app settings

Select an app to display that app’s settings and uninstall or stop using that app.

Settings app

Select in the upper right corner of the screen to display the details menu.

App permissions When installing an app, you can control which functions and information that app is allowed to access (this is called “permissions”).
App links If the system detects multiple apps which can perform the same operation (such as opening a connection or file), you will be prompted to select an app.
Advanced
Default apps Select the default browser or assistant and speech input apps.
Draw over other apps Choose which apps can be displayed over the other apps you are currently using.
Modify system settings Manage which apps are allowed to change system settings.
Battery optimization To conserve battery, manage the operating efficiency of various apps.

Details menu

Select the details menu in the upper right corner of the screen

Show system Display or hide system apps
Reset app preferences Remove app and background data limit settings. All settings will be reset.

Uninstalling apps

After selecting an app, the settings screen will appear. Select “uninstall”.
